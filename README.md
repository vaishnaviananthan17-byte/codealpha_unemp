# Unemployment Analysis in India using Python
# Project Overview

This project analyzes unemployment trends in India using Python.
The analysis includes data cleaning, visualization, regional unemployment comparison, rural vs urban analysis, and the impact of COVID-19 on unemployment rates.

# Technologies Used
Python
Pandas
Matplotlib
Seaborn
VS Code
Dataset

# Dataset Used: Unemployment in India.csv

The dataset contains:

Region
Date
Frequency
Estimated Unemployment Rate (%)
Estimated Employed
Labour Participation Rate (%)
Area (Urban/Rural)
